This is the DOCUMENTATION for myFlix

Hello, World! This is an application that will let you sort through different genres of movies
and directors, and let you save your favorite movies to your account.

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Allows a user to be created /users POST None A JSON object of the new user, structured like: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"} A JSON object of the new user, structured like: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"}
Returns a list of all movies /movies GET None None A JSON object of all the movies
Returns data about single movie by title /movies/:Title GET :Title None A JSON object a specific movie
Returns data about a genre by name /movies/genres/:Name GET :Name None A JSON object of a specific genre
Returns data about a director by name /movies/directors/:Name GET :Name None A JSON object of a specific director
Returns a list of all users /users GET None None A JSON object of all the users
Returns a list of a single user by username /users/:Username GET :Username None A JSON object of the specfic user
Allows users to change their information /users/:Username PUT :Username A JSON object of the new user, structured like: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"} A JSON object of the new user, structured like: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"}
Allows users to add a movie to their favorites /users/:Username/movies/:MovieID POST :Username, :MovieID None A JSON of the user
Allows users to remove a movie from their favorites /users/:Username/movies/:MovieID DELETE :Username, :MovieID None Text confirming the movie has been removed from the user's list
Allows a user to be deleted /users/:Username DELETE :Username None Text confirming the user has been deleted
